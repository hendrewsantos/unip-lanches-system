# Sistema de Lanchonete

Sistema web para gerenciamento de lanchonetes, com foco em controle de estoque e registro de vendas (PDV).

---

## Sobre

O projeto tem como objetivo facilitar a gestão do negócio, automatizando o controle de produtos, estoque e vendas em um único sistema.

---

- Spring Boot

- Spring Data JPA
- Spring Security
- JWT
- Hibernate
