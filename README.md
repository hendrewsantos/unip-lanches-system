# 🍔 Unip Lanches — Sistema de Gestão para Varejo Alimentício

Sistema full-stack desenvolvido para simular a operação de uma rede de lanchonetes, incluindo PDV, controle de estoque e gestão de produtos perecíveis.

---

## 🚀 Sobre o projeto

Este projeto foi desenvolvido como parte da Atividade Prática Integradora do curso de Análise e Desenvolvimento de Sistemas (ADS), com o objetivo de aplicar conceitos de engenharia de software em um cenário real.

O sistema contempla funcionalidades essenciais para o varejo alimentício, como:

- 🧾 Ponto de Venda (PDV)
- 📦 Controle de estoque
- ⏰ Monitoramento de produtos perecíveis
- 📊 Gestão de operações

---

## 💻 Tecnologias utilizadas

### Front-end
- React
- Vite
- Tailwind CSS

### Back-end
- Java 21
- Spring Boot

### Banco de Dados
- MySQL / H2

---

## 🖥️ Funcionalidades

### Front-end
- Interface de PDV com foco em performance e baixa latência  
- Dashboards e alertas visuais para gestão de estoque  
- Controle de acesso com rotas protegidas  
- Integração com APIs REST  
- Gerenciamento de estado com Context API  

### Back-end
- Autenticação e autorização com JWT  
- Implementação de regras de negócio  
- Gestão automatizada de estoque  
- Controle de produtos perecíveis  
- Arquitetura estruturada com Spring Boot
